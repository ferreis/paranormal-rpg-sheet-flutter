* [ ] Pensar em uma forma de sincronização sem precisar criar um servidor.
* [ ] Avaliar o uso de algum protocolo de broadcast ou conexão local.
* [ ] Permitir que o mestre crie um grupo.
* [ ] Permitir que o mestre compartilhe um código do grupo com os jogadores.
* [ ] Permitir que os jogadores entrem no grupo usando o código.
* [ ] Permitir que o jogador crie uma ficha do zero dentro do grupo.

* [ ] Mostrar para o mestre o que mudou entre as versões.
* [ ] Permitir que o mestre baixe a ficha atualizada do jogador.
* [ ] Esconder as opções de comparação quando as duas fichas forem idênticas.
